אפשרויות סידור התכנים בדרופל מגוונות.
אחת הדרכים היא אפשרות התיוג של פריטי תוכן  ואיגוד התיוגים בתוך מילון (taxonomy).

הוספת/עריכת תיוגים ( term ) :

באתר זה נעשה שימוש בסיווגים כדי ליצור קטגוריות שונות בתכנים מסוג : lectures & publications.
לסוג תוכן זה יצרנו מילון בשם type ובו 2 תגים.
כאשר מכניסים תוכן חדש מסוג זה יש לבחור בתג המתאים בשדה type.

כך בעצם נוצרת ההבדלה בין התכנים שבאה לידי ביטוי בתפריט הצדדי.

מנהל האתר יכול ליצור, לערוך, למחוק מילונים וסיווגים ללא הגבלה. למידע נוסף

הוראות לעריכת מילון:

  1. נעזר בתפריט המנהל השחור:
    structure -> taxonomy -> type
  2. בעמוד זה (דוגמה מילון type) ניתן לערוך, למחוק וליצור תיוגים חדשים
